CodingPuzzle

Learn programming in VR on Meta Quest

CodingPuzzle is a VR programming learning app for Meta Quest. Based on freeCodeCamp's curriculum, you learn by assembling code blocks in a puzzle format within VR space.

Available Courses

  • HTML
  • CSS
  • JavaScript
  • Python

Features

  • Puzzle-style code building in VR
  • Live HTML/CSS preview panel
  • Quiz-style questions for knowledge check
  • Multi-language support (14 languages)
  • Text-to-speech narration for instructions

Pricing

One step of puzzle-style coding is available for free every day.

With a subscription ($7.99/month or $59.99/year), you get unlimited puzzle-style coding with no daily limit.

All subscriptions include a 7-day free trial.

Start your VR programming experience today!

This project is not affiliated with or endorsed by freeCodeCamp.

How to Use

1. Select a Course

Choose from HTML, CSS, JavaScript, or Python on the course selection screen.

2. Select a Step

Each course contains multiple steps (workshops, lectures, quizzes). Select a step to begin.

3. Build Code

Code blocks float in front of you. Point at a block with your controller and press the trigger to place it in the correct position in the editor.

4. Check Your Work

For HTML/CSS courses, a live preview panel shows how your code renders. Complete all lines to advance to the next exercise.

Controls

  • Controller trigger or hand pinch: Select blocks
  • Thumbstick: Scroll panels
