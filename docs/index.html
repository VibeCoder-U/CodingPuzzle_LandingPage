CodingPuzzle

Build code with your hands — a new way to learn programming in VR/MR

CodingPuzzle is a VR/MR programming learning app for Meta Quest.

Available Courses

  • HTML
  • CSS
  • JavaScript
  • Python

Features

  • Puzzle-style code building in VR/MR using controllers or hand tracking
  • Live HTML/CSS preview panel
  • Quiz-style questions for knowledge check
  • Multi-language support (14 languages)
  • Text-to-speech narration for instructions

Pricing

One step of puzzle-style coding is available for free every day.

With a subscription ($7.99/month or $59.99/year), you get unlimited access to all courses and steps with no daily limit.

All subscriptions include a 7-day free trial.
